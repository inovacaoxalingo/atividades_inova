Configuração do Firebase — só precisa fazer 1 vez

1 Criar projeto no Firebase

Acesse console.firebase.google.com → clique em "Adicionar projeto" → dê um nome (ex: "teamboard") → pode desativar o Analytics → Criar projeto.

2 Ativar Realtime Database

No menu lateral: Criação → Realtime Database → Criar banco de dados. Escolha a região, selecione "Iniciar no modo de teste" e clique em Ativar.
